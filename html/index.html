Overview

This benchmark suite is aimed at the characterization of performance-oriented research targeting the POSIX shell. It consists of a collection of diverse shell programs collected from non-trivial tasks found out in the wild, accompanied by real inputs that fascilitate small and large scale performance evaluation.

This suite evolves to maintain its relevance. Please cite the version number associated with the release you are using.

The suite consists of 14 benchmarks. These span multiple domains, time periods, and interesting patterns that amend themselves to different optimizations and transformations.

Quick Setup

Koala can be obtained using the following ways:

  • Run curl up.kben.sh | sh from your terminal,
  • Clone the repo and run ./setup.sh,
  • Fetch a Docker container by running docker pull ghcr.io/kbensh/koala:latest, or
  • Build a Docker container from scratch.

More information in the README.
