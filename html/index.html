Overview

KOALA is a benchmark suite aimed at the characterization of performance-oriented research targeting the POSIX shell.

Quick Setup

PaSh has been run on Ubuntu, Fedora, Debian, and Arch. Use one of the following ways to set it up:

  • Run curl get.benchmarks.sh | sh from your terminal,
  • Clone the repo and run ./_infra/setup.sh,
  • Fetch a Docker container by running docker pull ghcr.io/binpash/benchmarks:latest, or
  • Build a Docker container from scratch.

More information in the README.
